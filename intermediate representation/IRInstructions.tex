\section*{IR Instructions}
All the instructions are defined by this data structure:
\begin{lstlisting}
type Intruction struct{
	op	int
	arg1	Symbol
	arg2	Symbol
	result	Symbol
	true	*Instruction
	false	*Instruction
	next	*Instruction
}
\end{lstlisting}
